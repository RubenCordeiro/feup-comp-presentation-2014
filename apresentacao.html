# Autoanalyze ### A Tool for Working with Automata Compilers Faculdade de Engenharia da Universidade do Porto
## What does it do?
* Reads automata from dotty files * Performs several analyzes and operations * Provides a simple language for specifying the operations
### Dot Automata ![A dotty file](images/dotfile.png)
## What's already been done?
* A DOT grammar parser * An intermediate representation for DOT graphs * A grammar for the new language AutoScript (AutomataScript)

Grammars

  • DOT Grammar
  • AutoScript Grammar
## What are we going to do?
* Implement automata analysis and operations * Extend transition labels to accept regular expressions * Methods to implement the automata in different languages * Elaborate the AutoScript language
